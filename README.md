## Overview

Apisonic Audio Speedrum is a Windows desktop utility made to change audio playback speed in real time while preserving pitch. It is intended for situations where fast, accurate control over listening or review is needed and where responsiveness matters.

The application is aimed at routine audio work that benefits from repeatable settings, quick capture, and modest resource use. Session profiles, user-defined hotkeys, and an optional launch-at-startup setting help make recurring tasks easier on supported Windows 10 and Windows 11 systems.
